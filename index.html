Thebe Lab

Thebe Lab is an experiment attempting to rebuild Thebe with javascript APIs provided by JupyterLab. This should make Thebe a smaller, more sustainable project.

Step by step demo

A minimal example

This page illustrates a minimal setup to get Thebe Lab running, using mybinder as kernel (i.e. computation backend) provider. See the source file for details.

This is a cell:

print("Hello!")

Press Shift-Enter to execute this cell. Launching the computation backend may take a few seconds, and you may need to re-execute it.

This is another cell, with plotting. Shift-Enter again! 

%matplotlib inline
import numpy as np
import matplotlib.pyplot as plt
x = np.linspace(0,10)
plt.plot(x, np.sin(x))
plt.plot(x, np.cos(x))
